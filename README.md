
# Graduation

졸업 작품으로 제작한 Polytech-Board 프로젝트를 정리하기 위해 만든 레포지토리입니다.

## 프로젝트 소개

Graduation 프로젝트는 학과 게시판의 응용 목적으로 제작된 프로젝트입니다. 기본적으로 대시보드의 형태를 가지지만 Gamepad를 이용한 조작이 가능하도록 제작되었습니다.

Gamepad를 이용하여 화면 전환이 가능하고 각 화면의 세부 정보를 확인할 수 있습니다.

화면의 구성은 다음과 같습니다.

- 메인 화면

프로그램이 시작하면 나타나는 화면입니다. 여러가지 Widget을 시각적으로 간단히 확인할 수 있고 정보는 선택 화면을 기준으로 표시됩니다.

- 선택 화면 (캘린더)

이번 달 캘린더를 집중적으로 볼 수 있는 화면입니다. 공휴일 또는 학사 일정을 확인 할 수 있습니다.

- 선택 화면 (카메라)

카메라 제어 화면입니다. Gamepad를 이용하여 카메라의 필터를 변경하고 사진을 촬영할 수 있습니다.

- 선택 화면 (커뮤니티)

워드클라우드의 세부 정보를 볼 수 있는 화면입니다. 워드클라우드에 메시지를 전송한 시간을 순차적으로 확인할 수 있습니다.
